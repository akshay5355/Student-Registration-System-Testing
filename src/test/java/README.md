# Student Registration System - Automation Testing

## Project Overview

This project automates the testing of a Student Registration System web application using Selenium WebDriver, Java, TestNG, and Maven.

## Features Tested

* Student Name Validation
* Email Validation
* Mobile Number Validation
* Gender Selection
* Department Selection
* Password Validation
* Confirm Password Validation
* Terms & Conditions Validation
* Registration Success Verification

## Technologies Used

* Java
* Selenium WebDriver
* TestNG
* Maven
* ChromeDriver

## Project Structure

* AppTest.java
* pom.xml
* testng.xml

## Learning Outcomes

* Automation Testing
* Test Case Design
* Bug Reporting
* STLC Concepts
* Selenium Framework Development
